# T1560.001 – Archive Collected Data | Solution Walkthrough
**Difficulty:** Easy | **OS:** Linux | **MITRE:** T1560.001

## Step 1 – Basic tar Archive
```bash
tar czf /tmp/loot.tar.gz /opt/loot/
ls -lh /tmp/loot.tar.gz
# Verify contents:
tar tzf /tmp/loot.tar.gz
```

## Step 2 – Password-Protected Zip
```bash
zip -P 'Ex!lPass' /tmp/loot.zip -r /opt/loot/
# Verify encrypted:
file /tmp/loot.zip
unzip -l /tmp/loot.zip
```

## Step 3 – 7zip with AES-256 + Header Encryption
```bash
7z a -p'Ex!lPass' -mhe=on /tmp/loot.7z /opt/loot/
# -mhe=on encrypts file headers too (hides filenames from inspection)
7z l /tmp/loot.7z  # Lists but won't show filenames without password
```

## Step 4 – OpenSSL Encrypted Archive
```bash
tar czf - /opt/loot/ | openssl enc -aes-256-cbc -k 'ExfilKey123' -pbkdf2 > /tmp/loot.enc
# Decrypt and extract:
openssl enc -d -aes-256-cbc -k 'ExfilKey123' -pbkdf2 < /tmp/loot.enc | tar xzf -
```

## Retrieve Flag
```bash
cat /opt/loot/documents/flag.txt
# OR from archive:
tar xzOf /tmp/loot.tar.gz opt/loot/documents/flag.txt
# HACKTIFY{ARCHIVE_EXFIL_<hash>}
```
