# T1048.003 – HTTP Exfiltration | Assessment

## MCQ 1
HTTP-based exfiltration via User-Agent or Cookie headers is preferred because:
A) It is faster  B) It blends with normal web traffic, making detection harder in proxy/firewall logs  ✅
C) It bypasses all DLP solutions  D) It requires no encoding

## MCQ 2
DNS exfiltration encodes data in:
A) DNS response bodies  B) Subdomain labels of DNS queries (e.g., data.attacker.com) that are sent to an attacker-controlled DNS server  ✅
C) DNS TTL fields  D) DNS MX records

## MCQ 3
DLP (Data Loss Prevention) solutions typically detect HTTP exfiltration by:
A) Blocking all HTTP traffic  B) Inspecting HTTP payloads for sensitive patterns (credit cards, SSNs) in outbound traffic  ✅
C) Monitoring DNS only  D) Checking file extensions

## Fill 1
`curl` flag to send an HTTP POST request:
**Answer:** `-X POST` (with `-d @filename` or `-d "data"`)

## Fill 2
Base64 flag to remove padding characters (useful for URL-safe encoding):
**Answer:** `tr -d '='` (remove `=`) combined with `tr '+/' '-_'`
