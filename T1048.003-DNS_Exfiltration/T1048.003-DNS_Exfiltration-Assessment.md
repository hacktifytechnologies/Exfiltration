A) DNS packets are undetectable  B) DNS queries to external servers are allowed by most firewalls and rarely inspected  ✅

## MCQ 2
In DNS exfiltration, data is encoded in which part of the DNS packet?
A) The DNS response body  B) The subdomain labels of the FQDN being queried  ✅
C) The DNS TTL field  D) The DNS type field

## MCQ 3
A detection rule for DNS exfiltration should alert on:
A) Any DNS query  B) DNS queries with subdomains longer than 50 characters or unusually high query frequency  ✅
C) All NXDOMAIN responses  D) DNS queries to port 53

## Fill 1
bash command to base64-encode a file for DNS exfiltration:
**Answer:** `base64 file.txt | tr '+/' '-_' | tr -d '='`

## Fill 2
DNS security mechanism that can block queries to known C2 domains:
**Answer:** `DNS RPZ (Response Policy Zones)` or `DNS sinkholing`
